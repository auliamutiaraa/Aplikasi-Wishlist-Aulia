# WishNote - Aplikasi Daftar Keinginan

WishNote adalah aplikasi web sederhana untuk mengelola daftar keinginan (wishlist) pribadi. Dibuat dengan HTML, CSS, dan JavaScript murni, aplikasi ini memungkinkan pengguna untuk menambah, mengedit, menghapus, dan melacak item-item yang diinginkan.

## âœ¨ Fitur Utama

- **Tambah Item**: Tambahkan item baru dengan nama, harga, kategori, prioritas, dan catatan
- **Edit & Hapus**: Modifikasi atau hapus item yang sudah ada
- **Status Pembelian**: Tandai item sebagai sudah dibeli atau belum
- **Pencarian**: Cari item berdasarkan nama
- **Filter & Sort**: Filter berdasarkan kategori dan status, urutkan berdasarkan berbagai kriteria
- **Statistik**: Lihat total item, item yang sudah dibeli, dan total nilai
- **Persistent Storage**: Data tersimpan di localStorage browser
- **Responsive Design**: Tampilan yang menarik dan responsif di semua perangkat
